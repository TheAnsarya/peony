## Work Done

### DisassemblyPipeline Class (Peony.Core)

Created `DisassemblyPipeline` static class with three helper methods:

- **`LoadHints()`** — Consolidates symbol/CDL/DIZ/Pansy loading into a single call. Returns null if no hint files exist.
- **`BuildEntryPoints()`** — Comprehensive, deterministic entry point merging from platform reset vectors + Pansy sub-entries + Pansy jump targets + Pansy cross-refs + CDL sub-entries. Uses ordered insertion for deterministic disassembly.
- **`CreateEngine()`** — Engine creation with symbol loader configuration (labels, comments, data definitions, static analysis toggle).

### Peony Disassembly Analysis

Ran `peony disasm` with Nexen CDL + Pansy on FFMQ:

- Result: 10,279 instructions, 38,246 .db lines, 6/16 banks with code
- Bank 0 entirely .db (where reset vector lives)
- CDL code coverage: 44,583/73,089 (61%)
- CDL data marked: 0/333,827 (0%)

### Bugs Found (5 root causes)

1. **Bank 0 mapping** - `AddressToOffset` treats bank 0 as unspecified
2. **Missing terminators** - `IsUnconditionalBranch` missing jml/rtl/bra/brl
3. **CDL JumpTargets** - 0x04 flag entries never seeded as entry points
4. **CDL label translation** - Raw ROM offset used as CPU address
5. **Bank propagation** - 24-bit address bank not extracted when queueing

### #158 — Add analyzer tests for untested platforms (commit 98c7488)

Created dedicated test files for 4 platforms with zero test coverage:

- **GenesisAnalyzerTests.cs** — SEGA header detection, SSFII mapper, exception vectors, register labels (VERSION, VDP_DATA, PSG, etc.), memory regions, address mapping
- **SmsAnalyzerTests.cs** — mapper detection (Sega/CodeMasters), bank switching (ld a,nn; ld(nn),a), Z80 entry points, register labels, memory regions
- **PceAnalyzerTests.cs** — copier header detection, HuCard/CD-ROM classification, TAM bank switching, vector reading from last page, MPR mapping
- **WonderSwanAnalyzerTests.cs** — ROM footer parsing (publisher, color/mono, orientation, SRAM), 70+ I/O port register labels, linear ROM mapping, OUT bank switching

NES and ChannelF already had comprehensive test coverage (discovered during audit).

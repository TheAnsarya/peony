## Architecture Documented

### 65SC02 CPU
- Enhanced 65C02 with new instructions
- New opcodes: BRA, PHX, PHY, PLX, PLY, STZ, TRB, TSB
- New addressing: Zero Page Indirect, Absolute Indexed Indirect
- Fixed JMP indirect bug

### Memory Map
- $0000-$00ff: Zero Page
- $0100-$01ff: Stack
- $0200-$fbff: Work RAM
- $fc00-$fcff: Suzy Registers
- $fd00-$fdff: Mikey Registers
- $fe00-$ffff: Boot ROM

### LNX Header Format
- 64-byte header with "LYNX" magic
- Page size, load/start addresses
- Game name (32 chars), manufacturer (16 chars)
- Rotation flag
